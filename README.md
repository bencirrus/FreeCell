# 🃏FreeCell

**FreeCell Solitaire in a browser for offline entertainment, built with Google AI Studio powered by Gemini.**

## ♥️ How to Play
 **Touch:**
- autoglides the card if it has a destination in the Tableau, or to a free cell if it doesn't.
- autoglides cards to foundation stacks.
- also undoes the move from a freecell.
- hint, undo a wasted sequence or redo it.

## ⚙️ Capabilities
- elderly friendly fluid gameplay. 
- autoscales to screen size.
- adopts day / night mode of the browser or adjust with the icon.
- load it before being out of pocket.
